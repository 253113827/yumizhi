# yumizhi.com - OpenClaw Skills Collection

欢迎来到 yumizhi.com 项目！这是一个汇集了超过1700个OpenClaw技能的综合平台，涵盖31个不同类别。

## 项目简介

yumizhi.com 是一个OpenClaw技能集合网站，提供以下功能：
- 搜索和浏览1700+ OpenClaw技能
- 按类别组织技能
- 中英文双语支持
- 技能下载功能
- 社区互动功能（开发中）

## 文件结构

- `openclaw_skills.html` - 英文版技能集合页面
- `openclaw_skills_cn.html` - 中文版技能集合页面
- `improved_openclaw_skills.html` - 增强版技能集合页面

## 技能分类

项目涵盖了以下主要类别：
- AI模型相关技能
- 开发工具技能
- 金融交易技能
- 云服务技能
- 安全技能
- 以及其他31个类别
